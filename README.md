# Walmart Store Sales Forecasting: XGBoost & Time Series Analysis

### 📊 Project Overview
This was the capstone project for my Data Science certification at **TS Academy**. 
My team focused on predicting weekly sales for Walmart stores using historical data, accounting for seasonal shifts and promotional markdowns.

### 🛠️ My Role: XGBoost Implementation
While the team explored various methods, I led the **XGBoost** implementation. Leveraging my background in **Mathematics (B.Tech)**, I focused on:
- **Feature Engineering:** Developing lag features and holiday-specific indicators to capture temporal dependencies.
- **Objective Function Optimization:** Tuning hyperparameters to minimize Root Mean Squared Error (RMSE).
- **Non-Linear Dynamics:** Capturing the complex relationship between CPI, Fuel Prices, and Store Sales.

### 🚀 Integration with my MSc Research
This project serves as a precursor to my work in **Embedded AI** at the University of Abuja. 
The next step for this model would be **Quantization** (INT8) to allow for real-time sales forecasting on handheld warehouse devices.
